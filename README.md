# Web App Inmobiliaria – QA

Proyecto de testing realizado sobre una aplicación web inmobiliaria desarrollada en un proyecto IT colaborativo basado en un caso real con cliente real.

## Rol
- Project Manager & Tester QA

## Alcance del trabajo
- Testing manual funcional
- Pruebas exploratorias
- Pruebas de frontend y backend
- Validación de endpoints mediante Postman
- Redacción de historias de usuario y criterios de aceptación
- Diseño y ejecución de casos de prueba

## Tipos de pruebas realizadas
- Pruebas funcionales
- Pruebas exploratorias
- Smoke testing
- Pruebas de API (Postman)

## Herramientas utilizadas
- Linear (gestión de tareas e incidencias)
- Postman (API testing)
- Excel (casos de prueba)
- Google Docs (documentación funcional)

## Casos de prueba
En este repositorio se incluyen **ejemplos representativos** de casos de prueba manuales utilizados durante el proyecto, con el objetivo de mostrar el enfoque de testing.

Los casos completos se encuentran documentados y están disponibles a solicitud.
